面试总结（一）

Author: 蒲笛扬 Date: July 18, 2019  13:38:00 Category: backEnd

面向对象：

  • 模板设计模式（多态）
  • 领域驱动设计
  • 集合类图

基础：

  • tcp 和 http的区别
  • 阻塞队列和非阻塞队列的区别
    • 阻塞队列与普通队列的区别在于，当队列是空的时，从队列中获取元素的操作将会被阻塞，或者当队列是满时，往队列里添加元素的操作会被阻塞
    • 阻塞队列的好处？
  • synchronized场景：
    • 防止重复执行
    • 依次执行
      • 依次刷新缓存
      • 锁住每个子账户

多线程：

  • 线程池的参数 *最大线程数 拒绝策略
  • 线程的状态 （4核8线程）running + ready = runnable
  • sleep -> time-waiting 线程状态变迁图
